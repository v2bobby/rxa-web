## 🛠️ Local Development
No build step required.

```bash
# Option 1: Open directly
open index.html

# Option 2: Use a local server (recommended)
npx serve .
# or
python3 -m http.server 8000
```

## 🚀 Deploy to Vercel (Recommended)
1. Push this folder to GitHub
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel will automatically detect the static site (thanks to `vercel.json`)

Or deploy instantly with Vercel CLI:
```bash
npm i -g vercel
vercel
```

## 🌐 Deploy to Other Platforms
- **Netlify**: Drag & drop the folder or connect GitHub repo
- **GitHub Pages**: Enable Pages on the repo root
- **Cloudflare Pages**: Connect GitHub repo
